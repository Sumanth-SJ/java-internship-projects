# Library Management System (Java)

## Project Overview
This is a console-based Library Management System developed using Java.
It follows Object-Oriented Programming (OOP) principles such as
Encapsulation, Abstraction, Classes, and Objects.

The system allows:
- Adding books
- Registering members
- Searching books
- Borrowing and returning books

## Technologies Used
- Java
- Collections (ArrayList)
- Scanner for user input

## How to Run
1. Open terminal / command prompt
2. Navigate to src folder
3. Compile:
   javac com/library/*.java
4. Run:
   java com.library.LibrarySystem
