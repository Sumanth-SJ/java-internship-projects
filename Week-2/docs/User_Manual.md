# User Manual

## How to Run
1. Open the project in a Java IDE
2. Navigate to `Week-2/src`
3. Run `GradeManagementSystem.java`

## Menu Options
1. Add Student Marks
2. View All Students
3. Calculate Averages & Grades
4. View Performance Report
5. Exit

## Input Rules
- Marks must be between 0 and 100
- Only numeric input allowed for marks
